  • What can be done if one develops a cataract after getting an implantable contact lens?

    OCT 21, 2018

    Question:

    What can be done if one develops a cataract after having an ICL (implantable contact lens) to cure myopia?


    Answer:

    Cataract removal can be scheduled electively and should be like a regular cataract surgery. The ICL will be removed and then the cataract is removed. The newly implanted artificial lens (IOL) replaces the ICL. If additional correction is needed, glasses, contact lenses or refractive surgery can finetune the refraction.


    Answered By: Jeffrey Whitman, MD
