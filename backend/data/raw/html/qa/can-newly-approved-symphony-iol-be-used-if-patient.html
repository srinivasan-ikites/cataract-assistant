  • Can the newly approved Symphony IOL be used if the patient has a macular pucker?

    NOV 20, 2016

    Question:

    Can the newly approved Symphony IOL be used if patient has a macular pucker?


    Answer:

    It can be used but it will not provide the maximum vision that a healthy retina would allow. For this reason, I would pick a single vision lens or an accommodative intraocular lens (IOL).


    Answered By: Jeffrey Whitman, MD

    Surgery Cataracts
